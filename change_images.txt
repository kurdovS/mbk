				change_images
				-------------

!Количество изображений для модуля должно быть >= 3!


1.change_images.inc - 	файл формирующий html-код для change_image. Для своей работы
			подключается к БД к таблице change_images (НУЖНО БУДЕТ
			ПОМЕНЯТЬ НАЗВАНИЕ ЧТОБЫ ИСПОЛЬЗОВАТЬ ДЛЯ РАЗНЫХ БРЕНДОВ).
			Считывает количество изображений, затем создает окно для них
			в нем - все полученные из БД изображения, затем toggle_circles,
			левую и правую кнопки-стрелки. Реализован как функция с 
			одноименным названием, возвращающая html-код как строку.


2.change_images.css - 	стили для change_images. Настраивают его внешний вид - 
			- change_images гибкий - подстраивается под любые разрешения.
			А также использует разные классы, которые переключаютя в JS,
			благодаря чему (смены классов) и происходит анимация.
			Анимация посредством смены значения свойства left с
			transition. Для окна задан position: relative - чтобы его
			потомки - изображения (имеющие position: absolute) - распола-
			гались относительно него и могли располагаться в одну строку
			находясь за пределами окна. Из-за того что предок (окно) не 
			расстягивается по потомкам с position: absolute (изображениям)
			высота окна высчитывается в JS. Потомки за пределами окна
			скрыты посредством overflow: hidden.


3.change_images.js - 	файл JS для change_images. 

			Имеет 10 глобальных переменных:
			из них 4 для получения элементов из html, 1 для числа изображ,
			2 для таймера и 3 главные переменные которые устанавливают,
			какое изображение слева от окна, какое в окне и какое справа.

			Имеет 10 функций: 1 для начальной инициализации, 1 для установ-
			ки высоты окна, 1 для получения кнопок-стрелок, 1 меняющую
			изображениям классы-стилей для анимации, 2 функции - обработ-
			чики нажатий на кнопки - они меняют номер изображения, которое
			должно быть в окне и вызывают функцию для смены классов-стилей,
			1 функция включающая и выключающая авто-смену изображений
			когда вкладка видна и не видна соответственно, 1 функция
			заглушка, нужная для того чтобы нельзя было нажать кнопки, во 
			время проигрывания анимации, 1 функция для того чтобы обратно
			включить кнопки, когда анимация завершилась - исп. как обработ-
			чик события - "transitionend" и 1 функция подсвечивающая
			нужный индикатор. 


4.change_images_index.php - 	сервисный файл - показывающий как использовать 
				change_images в своих проектах. в нем создается
				код страницы имитирующий рабочий код проекта, 
				подключается change_images.inc вставляющийся в код
				и страница выводится посредством echo.
